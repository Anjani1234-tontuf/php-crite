Features:
=========
a) The Sunset and Sunrise timings can be calculated easily.
b) The Script uses php combined with ajax which makes the faster retrival of the timings.
c) Provides the Sun-rise and Sun-set timings For large span.
d) Simple to integrate this page with any web site that supports php.

Requirements:
=============
Linux or Windows server (it *should* work on any server with PHP support) 
PHP 4.3.x or high


Installation:
=============
Please take 5 minutes time and read installation instructions carefully and
completely! This will ensure a proper and easy installation.

 1)Unzip HSunrise.zip.
 2) Just place include the code given in the "sunrise.php" file into your webpage and place the file "display.php" in the same folder.
 3) Set read,write permission to dispaly.php to create filter(only for linux users).
 4) Now call the "display.php" from your webpage after selecting the date and timezone, this may result the sunrise and sunset for the selected date.
